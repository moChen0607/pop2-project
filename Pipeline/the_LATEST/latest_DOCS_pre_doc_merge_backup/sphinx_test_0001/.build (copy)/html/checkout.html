class fileio.controller.checkout.FileCheckout(filePath)¶

An extension of the FileIO class, which now contains methods to determine whether or not a file has read/write access based upon certain criteria such as:

  1. If the name is incorrect (the regex to check names comes from the constants.model.constants module)
  2. If the file path isn’t too long
  3. If the file isn’t invalid in any way

Other verification methods like if the file is locked or if the file is in the correct folder or if the user is registered with the project can be found in the network module
