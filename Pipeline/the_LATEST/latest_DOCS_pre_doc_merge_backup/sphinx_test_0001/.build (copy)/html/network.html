network - Portal: Origins [Part 2]¶

All methods that require access and state information about the current network, will reference this package. It contains contextual information about the network and not about individual files specifically. If you’re looking for a lower-level access to file information, see: fileio - Portal: Origins [Part 2].

Portal Origins: Part Two

Project

Network module handles requests and changing information about the user and their access to the current network. Note, this file is not meant to be executed directly

class network.controller.network.NetworkManager[source]¶

Bases: object

The main interfacing class that is used to communicate over the network between projects/files/applications. It partially leverages the checkout module (See checkout) for general file I/O but extends upon it to make sure that files are OK and not locked/hidden/unsaveable/corrupted/etc
