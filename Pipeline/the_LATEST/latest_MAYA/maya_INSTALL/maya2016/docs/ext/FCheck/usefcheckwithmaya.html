Rendering utilities

Use FCheck with Maya

You can now launch FCheck from within Maya by selecting File > View Image or File > View Sequence.

Maya uses FCheck as an image previewer to display intermediate results of a rendering. FCheck can be opened within Maya when using these tools.

Playblast

    Uses FCheck to externally preview the current animation.

    See "Playblast" in the Animation guide on how to use FCheck with playblast.

Hardware Render Buffer

    Uses FCheck to externally display an animation rendering rendered using your computer's display graphics card.

    See "Use the hardware render buffer" in the Rendering guide on how to use FCheck with hardware render buffer.

Batch Render

    The "Show Batch Render" function uses FCheck to preview the latest rendered image when batch rendering.

    See "Show Batch Render" in the Rendering guide on how to open FCheck when batch rendering.
