Rendering utilities

Fcheck View menu (Windows)

Select to view specific channels from the View menu. By default, the display is set to All Channels.

The following is a list of channels available in the menu.

Red Channel

    Displays the red channel.

Blue Channel

    Displays the blue channel.

Green Channel

    Displays the green channel.

Alpha Channel

    Displays the alpha channel.

All Channel

    Displays RGB channels.

Luminance

    Displays the image in black and white.

Control Bar

    Toggles the control bar on and off.

Full Screen

    Displays the image or animation full screen.

Full Resolution

    Displays the image or animation at full resolution.
