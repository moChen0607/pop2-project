Rendering utilities

Image adjustment keys

Use the following keys to change the luminance, gamma, saturation or offset in images as well as change channel and manipulate colors.

Please note that an image must be written to disk to retain image modifications and adjustments executed in FCheck. See Save images or sequences for more details..

Windows/Linux

Mac OS X

Action

`

Resets all FCheck info window options.

d

d

Resets the image to the state it was in before you made changes to the luminance, gamma and saturation.

i

Displays the inverted bump; Similar to pressing n except that shows a negative gain.

l

Controls the luminance in an image, based on the weighted RGB. To toggle between line drawings and points, press L while holding down Shift.

n

Displays the normal bump. Increase depth by pressing ·and keys.

u

Updates or reloads the image.

w

Saves current image.

Shift + w

Saves entire animation.

=

}

Increase the luminance by 1%.

Shift + =

Increases the luminance by 10%.

Alt + =

Increases the luminance by 50%.

-

(keyboard, not numeric pad)

{

Decreases the luminance by 1%.

Shift + -

(keyboard, not numeric pad)

Decreases the luminance by 10%.

Alt + -

(keyboard, not numeric pad)

Decreases the luminance by 50%.

Ctrl + =/-

Pressing either Ctrl - or Ctrl + restores image to original luminance value.

]

]

Increases the gamma by 1%.

Shift + ]

Increases the gamma by 10%.

Alt + ]

Increases the gamma by 50%.

[

[

Decreases the gamma by 1%.

Shift + [

Decreases the gamma by 10%.

Alt + [

Decreases the gamma by 50%.

Ctrl + ]/[

Pressing either Ctrl ] or Ctrl [ restores image to original gamma value.

/

Increases the offset by 1%.

Shift + /

Increases the offset by 10%.

Alt + /

Increases the offset by 50%.

\

Decreases the offset by 1%.

Shift + \

Decreases the offset by 10%.

Alt + \

Decreases the offset by 50%.

Ctrl + \ or /

Restores image to original offset value.

`

`

Increases the saturation by 1%.

Shift + `

Increases the saturation by 10%.

Alt + `

Increases the saturation by 50%.

;

;

Decreases the saturation by 1%.

Shift + ;

Decreases the saturation by 10%.

Alt + ;

Decreases the saturation by 50%.

Ctrl + `/;

Pressing either Ctrl ` or Ctrl ; restores image to original saturation value.

F1

Switches to red bitplane channel control.

F2

Switches to green bitplane channel control.

F3

Switches to blue bitplane channel control.

F4

Switches to all colors (RGB) bitplane channel control. This is the default channel control in FCheck.

Shift + F1

Switches to cyan bitplane channel control.

Shift + F2

Switches to magenta bitplane channel control.

Shift + F3

Switches to yellow bitplane channel control.

F11

Allows images to be overwritten.
