FCheck Help menu (Windows)

Select an item to open a window which displays a number of help items, including keyboard controls, command line options, etc.

For information on commands, see FCheck.

The following is a list of choices available in the menu:

    • Opening/Saving Images
    • Animation Controls
    • Command Line
    • Mouse Controls
    • Keyboard
