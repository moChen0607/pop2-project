Rendering utilities

Supported file formats

The file format native to FCheck is Maya Image File Format (IFF). When a file is not in IFF format, FCheck uses an external converter called toimg to convert the images or animation. The conversion allows FCheck to display the image or animation in its playback window. The following is a list of FCheck supported file formats.
