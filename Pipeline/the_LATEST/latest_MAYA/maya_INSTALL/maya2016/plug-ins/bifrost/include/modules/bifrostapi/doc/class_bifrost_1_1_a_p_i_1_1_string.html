Detailed Description

Character strings in the Bifrost SDK API are represented with the String class. String is a thin wrapper around the standard library string class, though iterators are not exposed. The String class has also in-house methods for processing strings such as String::trim and String::split.

#include <bifrostapi/bifrost_string.h>
#include <iostream>
using namespace Bifrost::API;
String str( "\n\n\t \t 0123456789 \n" ) ;
// Trim blank characters
str.trimLeft( );
std::cout << str << std::endl;
// "0123456789 \n"

str.trimLeft( "42130" );
std::cout << str << std::endl;
// "56789"
// Trim " *** \t" characters
str = " *** \t0123456789";
str.trimLeft( " *\t" );
std::cout << str << std::endl;
// "0123456789"
