Abstract

The Bifrost Object Model (BOM) is an API library of hierarchical C++ classes that gives developers access to the Bifrost data model. It is aimed to create and manipulate Bifrost objects and data such as voxels and particles.

The library uses these kinds of classes:

  • API classes: these classes encapsulate the methods applicable to internal Bifrost objects. They expose the main access to Bifrost data and functionalities.
  • Core classes: provide some general services such as the generic class (Ref) used by the API classes for type introspection and object validation.
  • Math classes: these provide access to Bifrost math library elements such as vector, matrix and transformation.
