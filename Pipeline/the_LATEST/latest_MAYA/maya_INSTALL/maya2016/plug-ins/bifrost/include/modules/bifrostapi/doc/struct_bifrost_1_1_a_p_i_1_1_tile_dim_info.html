Detailed Description

TileDimInfo struct. Contains the meta data used for identifying the dimension values of a tile. In the current Bifrost architecture there are always 5x5x5 values in the tile (that's tileSize), the width of the tile (tileWidth) is the 5.

However, a tile can correspond to a larger region of tile space. At the finest level a tile covers 5x5x5, but at depth 6 it covers (25*25*25) and so on. The 25 is the depthWidth. This means that each voxel in a tile at depth 6 covers 5 units in the tile space. So the 5 is the voxelWidth.
