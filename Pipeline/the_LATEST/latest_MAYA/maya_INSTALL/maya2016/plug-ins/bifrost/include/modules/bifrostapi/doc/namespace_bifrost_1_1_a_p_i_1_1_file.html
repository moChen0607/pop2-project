Convert the input path separators to forward slashes. Note: The function assumes the path points to an existing file or folder. If the path points to an existing folder, an extra separator is appended if necessary. If the path is a folder path that doesn't exist, the path is normalized like a file path. The function doesn t validate the path for an existing file.

BIFROST_API_DECL String Bifrost::API::File::resolveFramePadding ( const String &  filePath,
int  frame 
)

Resolves a file path with padding for a specific frame value and returns a new resolved file path. The original path is returned if the input file path has no padding symbols.

Parameters
filePathFull file path. Frame position in the filename can be optionally specified with padding symbols like @ and #. The symbol # is a shortcut for @@@@.
frameFrame number (can be negative).
Returns
The full path properly formatted.
std::cout << "out: " << addPadding( "foo/whale_dad_jump-volume.@.bif", 1 );
// out: foo/whale_dad_jump-volume.1.bif
std::cout << "out: " << addPadding( "foo/whale_dad_jump-volume.@@.bif", 1 );
// out: foo/whale_dad_jump-volume.01.bif
std::cout << "out: " << addPadding( "foo/whale_dad_jump-volume.@@@.bif", 1 );
// out: foo/whale_dad_jump-volume.001.bif
std::cout << "out: " << addPadding( "foo/whale_dad_jump-volume.#.bif", 1 );
// out: foo/whale_dad_jump-volume.0001.bif
