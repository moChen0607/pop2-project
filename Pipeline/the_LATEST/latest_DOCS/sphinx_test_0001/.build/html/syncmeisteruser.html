The abs2rel script is syncs files from the network to local locations without user intervention. It also uses Absolute To Relative Paths under the hood to rename paths to be relative to the new copy location if the user chooses to enable that feature.

Important

Currently, I made the conscious decision to disallow this behavior. Instead of both syncmeister and abs2rel working at the same time, I ask that you run them separately. Parallel use is restricted to administrators, only.

The script was actually originally designed as a companion for R-E-D-A-C-T-E-D: You shouldn’t be here, but I figured that this tool was too useful to not make a quick GUI out of it (Hence why they both basically have the same GUI)
