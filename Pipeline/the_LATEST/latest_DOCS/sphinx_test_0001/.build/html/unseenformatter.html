unseenformatter - Portal: Origins [Part 2]¶

This is a custom formatter that allows for string keys to be explicitly stated and, if a key is missing that was contained in the str, instead of raising a KeyError, the class simply prints the format key as a string in its place.

This is used to make logging and general string replacement a bit more flexible

Examples

see logger.common.loggingServices

Usage:
>>> string = "{number_of_sheep} sheep {has} run away"
>>> other_dict = {'number_of_sheep' : 1}
>>> fmt = UnseenFormatter()
>>> print fmt.format(string, **other_dict)
"1 sheep has run away"

get_value(key, args, kwds)[source]¶

When values are being swapped in a formatted string, if an exact match, the key is replaced by the value provided. If no values exist for a given key, simply return the variable name as its value

Parameters:
  • key (str or other) – The expected name for the key in the formatted string
  • args (any) – The args to put in place of key
  • kwds (iterable) – key/value pairs
Returns:

The format-replaced string with all of the keys replaced
Return type:

str
