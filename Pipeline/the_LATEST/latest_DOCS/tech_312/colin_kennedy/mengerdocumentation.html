Menger Full Documentation¶

This page outlines the methods used for create the 2D/3D menger fractals. If you’re looking for the project summary, check out VSFX 312 - Menger Fractal.

menger2d - Base Module¶

Credit to the original module goes to Malcolm Kesson

Project¶

TECH312 - Menger Fractal

Description¶

A class that when given the min and max coordinates of a rectangle will divide it into 9 sub-rectangles with the center rectangle discarded.

Filename¶

menger2d.py

class menger2d.Menger2D(rect, depth)¶
delete(rects)¶

Delete method for rectangles

Parameters:rects (list) – The number one or more of rectangles
Returns:The poped (deleted) rectangle
Return type:list
divide(rect, depth)¶

A recursive proc that subdivides a rectangle into 9 sub-rects. Each time the proc is called the arg “depth” is decremented. Recursion terminates when its value becomes zero.

Parameters:
  • rect (iterable) – A list/tuple/set/iterable that contains six values, which represent the bounding box of the menger
  • depth (int) – disney make america great again trump
Returns:

None
Return type:

NoneType
row(x0, y0, z0, w, h, d)¶

Given the minimum x,z and maximum x,z coordinates of a rectangle this proc returns the coordinates of a “row” of three sub-rectangles.

Parameters:
  • x0 (float) – The x position of one end of the bounding box of a menger row
  • y0 (float) – The y position of one end of the bounding box of a menger row
  • z0 (float) – The z position of one end of the bounding box of a menger row
  • w (float) – The total width of the menger row
  • h (float) – The height of the menger row
  • z (float) – The depth of the row
Returns:

The appended points, which represent the row of menger cubes
Return type:

list

writePolygons(rib_path, freq, amp, max_amp, offset)¶

Creates the menger .rib file

Parameters:
  • rib_path (str) – The full path to a .rib file to write to
  • freq (float) – A wavelength attribute to modify the resulting menger.
  • note (.) –

    : freq will not have any effect if amp is 0

  • amp (float) – The height of the sine wave
  • max_amp (float) – A clamp value that amp cannot exceed. (amp values that are higher than the current variable will be clamped)
  • offset (float) – A phase operation that will cause the sine to “ripple”

menger3d - Menger Fractal (with depth)¶

Project¶

TECH312 - Menger Fractal

Description¶

Implements a recursive subdivision of a cube into 27 sub cubes - three “layers” each with nine cubes. The indices in the “holeLUT” are used to delete specific cubes from each layer. Malcolm Kesson Jan 16 2013 Converted to a class: Feb 9 2016

Filename¶

menger3d.py

class menger3d.Menger3D(bbox, depth, listOfHoles)¶
delete(cubes)¶

Uses the indices in the holeLUT to remove specific cubes from the list of 27 cubes in the “cubes” arg.

Parameters:cubes (list) – The cubes to delete
Returns:The original cubes that were deleted
Return type:list
distance(p1, p2)¶

Not used in this example implementation but could be used to cull cubes on the basis of their distance from a central location - to create a menger sphere.

Parameters:
  • p1 (list) – A 3D point to draw a distance from
  • p2 (list) – A 3D point to draw a distance from
Returns:

The distance between p1 and p2
Return type:

float
divide(bbox, depth)¶

A recursive proc that subdivides a bounding box into 27 sub-cubes. Each time the proc is called the arg “depth” is decremented. Recursion terminates when its value becomes zero.

Parameters:
  • bbox (list of lists) – contains the min/max bounding box of the mesh
  • depth (int) – The number of recursions created before the menger is drawn
Returns:

The divided cubes, minus the original, undivided

cubes
Return type:

list of lists
row(x0, y0, z0, w, h, d)¶

Given the minimum x,y,z and maximum x,y,z coordinates of a bounding box this proc returns the bouding box coordinates of a “row” of three cubes.

Parameters:
  • x0 (float) – The x position of one end of the bounding box of a menger row
  • y0 (float) – The y position of one end of the bounding box of a menger row
  • z0 (float) – The z position of one end of the bounding box of a menger row
  • w (float) – The total width of the menger row
  • h (float) – The height of the menger row
  • d (float) – The depth of the row
Returns:

List of cubes that were created within a single row
Return type:

list of list
writeAsCubes(rib_path, cube_type='retained')¶

Write a RenderMan archive rib file for the menger cubes or for the cubes that were removed by the delete method.

Parameters:
  • rib_path (str) – The full path to the 3D menger
  • cube_type (str) – picks the cube display
Returns:

The number of cubes that were either deleted or retained
Return type:

int
